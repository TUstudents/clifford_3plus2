# Theory

Spin(10) branching is textbook. The only new question is whether QCA supplies
the split and `J`.

## Textbook Part

For a fixed complex vector-space split

```text
V = C^3 ⊕ C^2
```

the even exterior algebra has dimension:

```text
S_+ = Lambda^even(V)
dim S_+ = 16
```

Counting selected basis directions in each block gives occupation numbers
`N_3` and `N_2`. Solving

```text
Y(0,0)=0
Y(0,2)=1
Y(1,1)=1/6
```

for `Y = A + B N_3 + C N_2` gives:

```text
Y = -1/3 N_3 + 1/2 N_2
```

The implemented branching table is:

| `(N_3, N_2)` | Multiplicity | `Y` | Label |
| --- | ---: | ---: | --- |
| `(0, 0)` | 1 | `0` | `nu^c` |
| `(0, 2)` | 1 | `1` | `e^c` |
| `(1, 1)` | 6 | `1/6` | `Q` |
| `(2, 0)` | 3 | `-2/3` | `u^c` |
| `(2, 2)` | 3 | `1/3` | `d^c` |
| `(3, 1)` | 2 | `-1/2` | `L` |

This is a representation identity. It is not evidence that QCA geometry
supplies the split.

## Load-Bearing Part

The bridge becomes nontrivial only if QCA data supplies all of the following:

1. Five candidate Clifford generators with a structural `3+2` block origin.
2. Exact Clifford anticommutation.
3. Compatible Euclidean signature, or a controlled Wick-rotation origin.
4. An explicit complex structure operator `J`.
5. `J^2 = -I` exactly.
6. `[J, P3] = 0` and `[J, P2] = 0` exactly.
7. `J` generated by or included in the allowed QCA gate algebra.
8. Allowed gates in the SM commutant, not just block diagonal.

Until those conditions are met by exact input data, the verdict remains
`notation_only`.

## Gate Algebra Boundary

The safe one-particle gate algebra is not all of

```text
gl(V_3) ⊕ gl(V_2)
```

For the audited one-particle split, safe generators are block scalars:

```text
lambda_3 I_{V_3} ⊕ lambda_2 I_{V_2}
```

For the even Fock representation, safe diagonal invariants may be functions of
`N_3` and `N_2`. A generator can be block diagonal and still fail if it resolves
individual color-basis directions inside `V_3`.
