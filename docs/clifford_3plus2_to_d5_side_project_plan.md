# Side Project Plan: 3+2 Clifford Geometry To D5 / Spin(10)

**Document type:** theory and roadmap plan  
**Project context:** BCC-QCA / geometry-to-internal-symmetry bridge  
**Status:** side project proposal; no theorem claimed yet  
**Date:** 2026-05-13

## 0. Purpose

The main BCC-QCA stream has a missing bridge:

```text
physical QCA geometry -> internal D5 / Spin(10) one-generation structure.
```

The direct bridge

```text
BCC body diagonal sector -> D5 Cartan weight projector
```

was falsified because it breaks continuous color.  The new side-project idea
is different:

```text
QCA Clifford geometry supplies a 3+2 decomposition.
D5 / Spin(10) uses the same 3+2 decomposition internally.
The bridge is through invariant Clifford structure, not Cartan projectors.
```

The target is a theorem-level statement explaining why a local QCA with
spatial chirality plus a wall/Floquet mass plane naturally produces the
internal algebraic skeleton of one Standard-Model generation:

```text
C^5 = C^3 ⊕ C^2
Spin(10) spinor 16 = Lambda^even(C^5)
Y = -1/3 N_3 + 1/2 N_2
```

This side project does **not** claim to explain three families.  It aims to
explain why a single `Spin(10)` `16` is the natural internal fiber associated
with the QCA Clifford data.

Important scope warning:

```text
The exterior-algebra branching and the N_3,N_2 commutant statements are
standard Spin(10)/GUT bookkeeping.
```

They are necessary notation checks, not the new theorem.  The only genuinely
load-bearing claim is:

```text
the QCA supplies the 3+2 Clifford split structurally.
```

If the two extra Clifford directions are arbitrary coin decorations or already
internal labels, the side project collapses to notation and should be archived
as such.

## 1. Core Hypothesis

The QCA supplies two geometrically different Clifford blocks:

```text
3 spatial Weyl/QCA directions
2 wall/Floquet/mass-plane directions
```

Here "wall/Floquet/mass-plane" is only shorthand.  It is not yet a defined
object.  Phase 0 must decide which, if any, of the following candidates is
real:

```text
wall normal + real-time/Floquet direction
Euclideanized wall/Floquet pair
tau_x,tau_y coin plane
some other structural QCA two-plane
```

Until that audit passes, the two-plane is an unknown, not an assumption.

Together these define a stable `3+2` Clifford split:

```text
V_5 = V_3 ⊕ V_2.
```

After complexification and second quantization, the internal spinor
construction gives:

```text
S_+ = Lambda^even(C^5)
dim S_+ = 16.
```

This also hides a second hard datum: a complex structure.  The standard
Spin(10) spinor construction is naturally built from a maximal isotropic
complex space:

```text
R^10 with complex structure J
-> C^5
-> S_+ = Lambda^even(C^5).
```

The `3+2` split used for Standard-Model branching is really the familiar
intermediate step:

```text
Spin(10) ⊃ SU(5) ⊃ SU(3) x SU(2) x U(1).
```

So the side project must explain where the complex structure `J` comes from.
Possible origins:

```text
QCA chirality
Floquet phase / microscopic complex phase
Euclidean Wick rotation
chosen by hand
```

If `J` is chosen by hand, the bridge is not canonical.

The Standard-Model subgroup is the stabilizer of the `3+2` split:

```text
SU(3) x SU(2) x U(1) subset Spin(10),
```

with hypercharge generated by the number operators on the two blocks:

```text
Y = -1/3 N_3 + 1/2 N_2
```

up to a conventional additive normalization fixed by the chiral spinor sector.

The geometry does not label red, green, blue, or weak up/down states
individually.  It labels only the block dimensions:

```text
3 and 2.
```

That is why this route may preserve continuous `SU(3)c` and `SU(2)L`.

## 2. What Would Count As A Theorem

The desired theorem is not:

```text
BCC directions are D5 Cartan axes.
```

That route is dead.

The desired theorem is:

```text
Given QCA Clifford data consisting of a spatial 3-plane and an independent
wall/Floquet 2-plane, there is a canonical Spin(10) spinor construction whose
3+2 stabilizer reproduces exactly one Standard-Model generation, and whose
allowed geometric operations act by intertwiners rather than color-breaking
projectors.
```

More precise theorem target:

```text
Let V = V_3 ⊕ V_2 be a complex five-dimensional vector space with
dim V_3 = 3 and dim V_2 = 2.  Let S_+ = Lambda^even(V).

Define N_3 and N_2 as occupation number operators on V_3 and V_2.
Then the decomposition of S_+ under
S(U(V_3) x U(V_2)) = SU(3) x SU(2) x U(1)
is exactly:

  (1,1)_0
  (1,1)_1
  (3,2)_{1/6}
  (bar3,1)_{-2/3}
  (bar3,1)_{1/3}
  (1,2)_{-1/2}

after choosing the standard hypercharge normalization.
```

The QCA-specific theorem adds:

```text
The QCA Clifford split and complex structure determine V_3 and V_2 up to
block rotations only.
Therefore geometric gates invariant under the split commute with the full
SU(3) x SU(2) x U(1) algebra and do not resolve Cartan weights.
```

Hidden-data warning:

```text
If the complex structure J or the SU(5) embedding is chosen independently of
the QCA, the construction is notation_only rather than a QCA-to-D5 bridge.
```

## 3. Why This Might Be Valuable

If proved, this would establish:

```text
geometry -> one-generation Spin(10) representation
```

without:

```text
geometry -> color projector
```

It would give the BCC/QCA program a defensible internal-symmetry bridge:

```text
physical QCA geometry supplies a 3+2 Clifford split;
D5 supplies one SM generation;
family number remains a separate boundary/defect/Floquet problem.
```

This is a smaller claim than deriving the full Standard Model, but it is a
real theorem-shaped claim.

## 4. Objects To Define

### 4.1 QCA Clifford Data

Define the five Clifford generators or planes that arise from the QCA:

```text
spatial block:
  e_1, e_2, e_3

wall/Floquet mass block:
  m_1, m_2
```

They should satisfy:

```text
{e_i,e_j} = 2 delta_ij
{m_a,m_b} = 2 delta_ab
{e_i,m_a} = 0
```

or the corresponding complex/anti-Hermitian convention.

The first deliverable must identify these objects concretely in the existing
QCA language:

```text
spatial Weyl Pauli/Clifford directions
candidate wall/Floquet/mass-plane directions
chirality operator / grading
```

The two-plane is the hard part.  The project must decide whether the two
generators are:

```text
1. geometric directions from wall normal + time/Floquet phase,
2. Euclideanized wall/Floquet directions after Wick rotation,
3. coin-sector matrices tau_x,tau_y,
4. arbitrary tunable internal decorations.
```

These are not equivalent.  If the two-plane has Lorentzian signature `(1,1)`,
the Euclidean `C^5` spinor construction needs an explicit Wick-rotation or
complexification argument.  If the two-plane is merely `tau_x,tau_y` in an
internal coin sector, then the bridge from geometry to internal symmetry is
not established.

The project must therefore prove:

```text
mass_plane_structural: true
mass_plane_signature_matched_to_C5: true
mass_plane_not_arbitrary_internal_decoration: true
```

It must also identify the complex structure:

```text
complex_structure_origin:
  qca_chirality
  floquet_phase
  wick_rotation
  by_hand
  unknown
```

Passing requires:

```text
complex_structure_origin != by_hand
complex_structure_origin != unknown
complex_structure_compatible_with_3plus2_split = true
```

### 4.2 Internal Fermionic Oscillator Construction

From the complex five-dimensional space:

```text
V = C^5 = C^3 ⊕ C^2,
```

introduce fermionic creation operators:

```text
a_i^\dagger, i=1,2,3
b_alpha^\dagger, alpha=1,2.
```

The chiral Spin(10) spinor is:

```text
S_+ = span of even-occupation states.
```

Number operators:

```text
N_3 = sum_i a_i^\dagger a_i
N_2 = sum_alpha b_alpha^\dagger b_alpha.
```

Hypercharge candidate:

```text
Y = A + B N_3 + C N_2.
```

Solve `A,B,C` so the even exterior algebra states match one SM generation.

### 4.3 Gauge Algebra

Define:

```text
su(3) acts on C^3
su(2) acts on C^2
u(1)_Y acts by Y
```

Then compute the branching of `S_+`.

The key check:

```text
block rotations preserve multiplets;
Cartan projectors are not used.
```

### 4.4 Combined Geometric Operations

A geometric operation `g` should act diagonally:

```text
psi(x) -> Omega(g) psi(g^{-1} x)
```

where:

```text
g acts on QCA Clifford data,
Omega(g) acts as the induced block-preserving Spin(10) or automorphism action.
```

Allowed operations must be intertwiners:

```text
Omega(g) rho_SM(X) Omega(g)^-1 = rho_SM(phi_g(X))
```

or commute with the SM algebra when the operation is meant to be a spacetime
symmetry only.

Forbidden:

```text
operators that select individual D5 Cartan weights inside color multiplets.
```

### 4.5 Block-Invariant Gate Algebra Constraint

The bridge requires a restriction on the allowed local QCA gate algebra.

It is not enough that the abstract vector space decomposes as:

```text
C^5 = C^3 ⊕ C^2.
```

If local gates can address individual modes inside the `C^3` block, they can
recreate the killed color-breaking projector route.  The allowed gate algebra
must be closed under block rotations:

```text
U(3) x U(2)
```

or, after imposing determinant/hypercharge conventions:

```text
SU(3) x SU(2) x U(1).
```

Hard hypothesis:

```text
Allowed QCA local gates are generated by block-invariant tensors built from
the QCA Clifford split, not by projectors onto individual C^3 or C^2 basis
vectors.
```

This also forbids block mixing.  The allowed infinitesimal gate algebra must
lie in:

```text
gl(V_3) ⊕ gl(V_2)
```

with the appropriate unitary/orthogonal/reality conditions, not in the full:

```text
gl(V_3 ⊕ V_2).
```

Equivalently, the off-block components must vanish:

```text
Hom(V_3,V_2) = 0,
Hom(V_2,V_3) = 0
```

for the allowed geometric/internal gate generators.

Falsifier:

```text
If the QCA rule set permits generic basis-addressing local gates inside C^3,
then the bridge does not protect color and Phase 3's commutant lemma is
irrelevant.
```

Sibling falsifier:

```text
If the QCA rule set permits geometric generators that mix V_3 and V_2, such as
off-block bilinears e_i m_a, then the SM subgroup is not protected.
```

## 6. Concrete First Calculation

The textbook branching calculation is useful, but it is not the first
load-bearing calculation.

Do this first:

```text
Identify the actual QCA candidate two-plane and determine whether it is
structural, geometric, and signature-compatible with the C^5 construction.
```

Minimum output:

```text
candidate_generators: [...]
anticommutation_matrix: [...]
signature: Euclidean / Lorentzian / mixed / unknown
structural_origin: wall-normal, Floquet-time, coin-sector, arbitrary
complex_structure_origin: qca_chirality / floquet_phase / wick_rotation / by_hand / unknown
off_block_gate_generators_present: true/false
block_invariant_gate_algebra: true/false
verdict: structural_bridge / notation_only / falsified
```

Then do the notation check:

```text
Compute the exterior-algebra branching of S_+ = Lambda^even(C^3 ⊕ C^2)
under SU(3) x SU(2) x U(1)_Y.
```

Use states:

```text
|I,J>, I subset {1,2,3}, J subset {4,5}, |I|+|J| even.
```

Let:

```text
N_3 = |I|
N_2 = |J|.
```

Find constants `A,B,C` in:

```text
Y = A + B N_3 + C N_2
```

so the six sectors match:

```text
(N_3,N_2)=(0,0) -> nu^c, Y=0
(0,2) -> e^c, Y=1
(1,1) -> Q, Y=1/6
(2,0) -> u^c, Y=-2/3
(2,2) -> d^c, Y=1/3
(3,1) -> L, Y=-1/2
```

Expected solution:

```text
Y = -1/3 N_3 + 1/2 N_2.
```

This is the first hard check because it proves the 3+2 split really contains
the one-generation hypercharge pattern.

## 7. Falsifier Ledger

The side project fails if any of these occur:

```text
1. The 3+2 exterior-algebra branching does not reproduce one SM generation.

2. The QCA does not naturally supply a 3+2 Clifford split.

3. The wall/Floquet mass plane is arbitrary and not structural.

4. The combined geometric operations require projectors onto individual color
   weights.

5. The construction breaks SU(3)c or SU(2)L when made local.

6. The theorem reduces to notation: "given Spin(10), recover Spin(10)"
   without explaining why QCA geometry suggests C^3 ⊕ C^2.

7. Phase 2 and Phase 3 pass, but Phase 0/4 fail.  This means the project has
   only reproduced textbook Spin(10) branching and must not be integrated as
   a geometry-to-D5 result.

8. The allowed QCA gate algebra is not block-invariant and can address
   individual basis directions inside `C^3` or `C^2`.

9. The complex structure `J` needed for `C^5` and the `SU(5)` intermediate
   embedding is chosen by hand.

10. The allowed QCA gate algebra contains off-block generators in
    `Hom(V_3,V_2)` or `Hom(V_2,V_3)`.
```

## 8. Claim Boundary

If successful, this side project proves at most:

```text
The QCA 3+2 Clifford geometry naturally matches the internal D5/Spin(10)
one-generation spinor construction.
```

It does not prove:

```text
three families
mirror gap
Golterman-Shamir inertness
continuum gauge dynamics
Higgs/Yukawa realism
phenomenology
```

The bridge is still valuable because it would answer a narrower but important
question:

```text
Why Spin(10) 16 here?
```

without overclaiming:

```text
Why the full Standard Model?
```
