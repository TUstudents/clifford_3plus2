## Writing Standard

The synthesis text is Markdown with LaTeX formulas using `$...$` and `$$...$$`.
It should read as theory prose, not as a Python script, checklist, or verdict
catalog. Bookkeeping appears only when it clarifies the physics.

Each claim carries a certainty label:

| Level | Meaning |
|---|---|
| `C:9` | Proven theorem or exact symbolic identity under the stated definitions. |
| `C:8` | Exact finite certificate or exhaustive computation with decisive controls. |
| `C:7` | Derived result inside the accepted construction, with ordinary assumptions explicit. |
| `C:6` | Conditional theorem depending on a named nontrivial hypothesis. |
| `C:5` | Strong organizing equivalence or duality, not yet a theorem. |
| `C:4` | Compatible reconstruction or inverse realization, not forward derivation. |
| `C:3` | Plausible conjectural mechanism needing a proof or falsifier. |
| `C:2` | Weak speculation or bookkeeping correlation with little evidential force. |
| `C:1` | Falsified or killed by a decisive control. |

- CP/phase holonomy and killed generation routes.

The central discipline is to identify the primitive mathematical object first:
graph Laplacian, nilpotent flag, $S_3$ shell, Schur complement, projector, or
spectral measure. The physics interpretation comes after the object is pinned
down.
