# cusp

Sidecar for the CUSP-PLAN flavor program.

The source plan is [CUSP-PLAN.md](CUSP-PLAN.md).  The working proposal is:

```text
flavor hierarchy = Froggatt-Nielsen grading from a (2,3) recirculation cusp
```

## Session 01 - Targets A-D

Implemented in [targets.py](targets.py):

- **Target A**: the local BCC edge q-clock has two q-preserving same-normal
  branches, `(sigma1,sigma2)=(+,+),(-,-)`, and two q-changing leakage branches,
  `(+,-),(-,+)`.  The exact BB edge blocks have the half-plus-half norm split
  `same-normal = I/2`, `mixed-normal = I/2`; stacking the same/mixed outgoing
  blocks gives a local isometry with finite unitary completion.  A q-mismatch
  hard-gap/outgoing-leakage audit makes mixed-normal feedback vanish, while a
  recurrent leakage control has a nonzero two-step return and is rejected.  The
  q-preserving pair supplies the weak/BCC parity clock with order and
  multiplicity `2`; the color `SU(3)` center supplies the primitive closure
  `3`.  The finite
  recirculation graph is generated by local center-charge automata: each tick
  advances the fundamental center charge, and the visible sheet reads out only
  charge `0`.  The weak automaton has charge table `(0,1,0,1,...)` and
  primitive return `2`; the color automaton has `(0,1,2,0,...)` and primitive
  return `3`.  This gives two length-2 weak returns and three length-3 color
  returns.  Enumerating closed walks recovers primitive lengths `(2,3)`,
  valuation algebra `C[t^2,t^3]`, first valuations `(0,2,3)`,
  and gap `1`; controls `C[t]`, `C[t^2]`, `C[t^3]`, a one-step visible loop,
  weak-only, color-only, and wrong-color-length finite graphs give different
  low valuations.
- **Target B**: the normalized color/weak shear is the unique solution and
  stable minimum of the squared mismatch functional for
  `(1+t)sqrt(2)=sqrt(3)`, giving `lambda_rec = sqrt(3/2)-1`; the ordinary
  reflection coefficient is identified as the solution of a different
  two-sided scattering control, and is therefore excluded by the one-sided
  retarded recirculation boundary.  Count-ratio shear and inverse-amplitude
  shear also fail the one-sided norm-matching equation.
- **Target C**: SM hypercharge conservation forces the up Yukawa to use
  `H_tilde` and the down Yukawa to use `H`; swapped doors are rejected.  Given
  that door orientation, the semigroup `S=<2,3>` derives conductor `c=2` and
  Frobenius gap `1`.  The down right charges are now selected as the conductor
  ideal residue `D=max(Q-c,0)=(1,0,0)`, and the up lift factor is the weak/BCC
  primitive closure order `2`, giving `U=(5,2,0)`.  Wrong-conductor, trivial
  lift, and color-order lift controls miss the exponent skeleton, and a finite
  solver confirms these charges are the unique nonnegative solution for the
  CUSP-PLAN diagonal exponent targets.  The deeper microscopic origin of the
  conductor-module and weak-double-cover boundary dynamics is still open.
- **Target D**: coefficients are built as finite path sums
  `c_ij=sum_gamma A_gamma Omega_gamma` with `Omega_gamma` in the color center.
  The up center powers are selected as geodesic distances on the non-cyclic
  length-3 cusp flag, and the down powers are selected as the unit bilinear
  pairing on `F3` center labels.  The rule-pair witness has nonzero
  `Im tr([YuYu^dagger,YdYd^dagger]^3)`.  The all-real, one-sector, and
  separable row/column controls give zero, and independent
  common-left/up-right/down-right field rephasings leave the invariant
  unchanged.  Session 09 replaces the arbitrary positive path-amplitude
  deformation with the cusp-module count rule
  `A_ij=max(1,# decompositions of q_i+r_j in <2,3>)`; the gap exponent `1` is
  one irreducible conductor-module/contact path.  This derived positive
  amplitude measure remains CP-active while the corresponding all-real control
  is CP-zero.  The remaining witness part is no longer finite topology
  selection, but the microscopic BCC boundary-material derivation of that
  topology.

## Session 02 - Local Center-Charge Source

Implemented in [targets.py](targets.py) and
[session_02_center_source.py](scripts/session_02_center_source.py):

- replaces the arbitrary fixed-length Target A paths with a local
  `CenterChargeRecirculationAutomaton`;
- one fundamental recirculation tick carries nontrivial center charge, so a
  one-step visible readout is forbidden;
- neutral visible return first occurs at order `2` for the weak/BCC doublet and
  order `3` for the color triplet;
- the generated automaton graph has the same closed-walk counts and primitive
  generators as the Target A cusp graph.

This is still not the full microscopic BB boundary-material theorem.  It is the
finite local center-charge model that the later BCC boundary construction has
to realize.

## Session 04 - Boundary Dilation

## Session 05 - Boundary-Material Origin

Implemented in [targets.py](targets.py) and
[session_05_boundary_material_origin.py](scripts/session_05_boundary_material_origin.py):

- derives the locking penalty under finite local material axioms: lowest-degree
  q-local polynomial, q-reflection symmetry, positivity on adjacent leakage
  sectors, and zero penalty on q=0;
- solves the constraints exactly and selects `p(q)=g q^2`, giving gap `4g` on
  `q=+-2`;
- rejects the linear control because it splits `q=+2` from `q=-2`;
- rejects the constant control because it gaps the visible q=0 scar;
- derives the outgoing closure from the no-incoming causal rule and keeps the
  recurrent mixed-normal closure as the nonzero-return control.

This reduces the material assumption to two named physical axioms:
q-local positive q-reflection stiffness and no-incoming retarded asymptotics.
Those axioms still need a deeper BB/QCA boundary-material realization.
