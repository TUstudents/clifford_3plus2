## E1/E2 Negative Proposition

Let `A_R` be the real algebra generated by a searched primitive family `R`.
Assume the generators are built only from the coarse `SO(6) x SO(4)`-symmetric
operations used in E1/E2: global clock, whole-block reflections, and
permutations internal to the already declared blocks. Then `Z(A_R)` contains no
new central rank-`(6,4)` idempotent pair. The only nontrivial coarse pair that
appears is the already supplied `(P_3, P_2)` pair, for example through the
whole-block reflection.

Proof: the fully coarse generators preserve too much within-block symmetry to
select a `6+4` central idempotent, while any primitive that does select the
`3` block versus the `2` block has already supplied the coarse split; adding
rank-one controls supplies lower-rank central idempotents and is rejected by
the no-locking falsifier.

Consequence: increasing only the word depth or coefficient search bounds in
this primitive class is not a route to the missing object. A positive search
must add a physical source of `3` versus `2` asymmetry before projector
discovery, such as Floquet polarization or defect monodromy.
